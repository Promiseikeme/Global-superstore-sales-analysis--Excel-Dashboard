# Global Sales Analysis
This project presents an interactive sales analysis dashboard built using Microsoft Excel, Power Query, and Pivot Tables. The analysis provides insights into sales performance, and shipping trends across different regions and product categories.
The goal of this project is to demostrate data cleaning, transformation, aggregation, and visualisation techniques used in business intelligence and data analytics.

**Tools used** 
Microsoft Excel
Power Query for data cleaning and Transformation
Pivot Tables for aggreggation and analysis
Pivot Charts for Visualization
Slicers For interactive filtering

**Dataset** The dataset contains contains global sales transactions with the following field Order Id, Customer Id, product category, market/ Country, sales, profits, Quantity, Discount,shipping mode, Order priority Year. The dataset was transformed using power query before being loaded into Excel for analysis
**KPIs ** Total Sales $12,642,905 Total Profit: $1,467,457, Total quantity sold: 178,312, Average Discount: 0.081
Key Visualization
Sales by Category , shows the distribution of sales across product categories, Profitability displays profitsbility across customer segment. Sales by market highlights the performance of different market. Impact on discount examines the relationship between discount levele and profitability.

**Insights** Technology category generated the highest sales revenue, the Cosumer Segment contributed to share of the profit. Standard Class is the most freqently used shipping mode. Sale shoowssteady growmth from 2011 to 2014. Higher discount tends to reduce profit.
